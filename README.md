# Shopify Scripts for Japan
日本国内向けECで使えるShopify Scripts集


## Shopify Scriptsの制約

- Line Items、Shipping、Paymentスクリプトと３種類ありますが、同じ種類のスクリプトは1つまでしか有効化できません
- Shopify ScriptsでLine Itemの追加はできません（ノベルティ付与目的など）
